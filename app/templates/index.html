SmartLowPicker 股票分析平台

基于 AKShare 和 Flask 构建的 Python 后台服务

项目简介

SmartLowPicker 是一个用于股票策略分析与选股的后端服务，使用 AKShare 获取财经数据，结合 Pandas 处理数据，并通过 Flask 提供 Web API 接口。

功能特性

  • ✅ 股票行情、财报、估值指标自动获取
  • 📊 使用 Pandas 快速处理并筛选数据
  • 📈 可视化输出选股结果（支持 Matplotlib）
  • 🧠 易扩展、可集成至前端平台或量化系统

使用方式

本项目为服务端 API，启动方式如下：

python run.py

接口如：GET /api/stock/summary 返回选股结果
